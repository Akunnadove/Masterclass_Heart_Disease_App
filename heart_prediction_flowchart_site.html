Heart Disease Prediction Visualizer

Follow the steps to understand how we predict heart disease using Python!

Start Project 🎬
⬇️
Import Dataset 📁
df = pd.read_csv("heart.csv")
⬇️
Explore Data 🔍
df.head()
df.describe()
⬇️
Split Features & Target 🧪
X = df.drop("target", axis=1)
y = df["target"]
⬇️
Scale the Data 📏
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
⬇️
Train the Model 🧠
model = LogisticRegression()
model.fit(X_scaled, y)
⬇️
Evaluate the Model 📊
print(classification_report(y, model.predict(X_scaled)))
